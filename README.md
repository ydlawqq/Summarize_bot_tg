# Телеграмм бот

Обычное общение происходит через LLM.
Пользователь в любой момент может загрузить файлы (только PDF).

Также пользователи могут искать данные в своих файлах — для этого достаточно нажать на кнопку **"Поиск по документам"**.

---

## Инициализация приложения
- FastAPI

## Основные фреймворки 
- LangChain
- LangGraph
- LlamaIndex
- SQLAlchemy
